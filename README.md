# 🎓 CodeAlpha Internship Tasks

This repository contains the machine learning and data science tasks completed during my internship at **CodeAlpha**. These projects demonstrate the application of various algorithms to solve real-world problems.

---

## 📂 Projects Overview

### 1. Diabetes Prediction
- **File**: `diabetes_prediction.py`
- **Description**: A predictive model developed to determine the likelihood of diabetes in patients based on medical diagnostic measurements.
- **Key Concepts**: Data preprocessing, Exploratory Data Analysis (EDA), Classification algorithms.

### 2. MNIST Digit Classification (CNN)
- **File**: `mnist_cnn.py`
- **Description**: A Convolutional Neural Network (CNN) built to classify handwritten digits from the famous MNIST dataset.
- **Key Concepts**: Deep Learning, Neural Networks, Computer Vision, TensorFlow/Keras.

---

## 🛠️ Technologies Used
- **Language**: Python
- **Libraries**: 
  - `Pandas` & `NumPy` for data manipulation.
  - `Scikit-learn` for traditional machine learning models.
  - `TensorFlow` / `Keras` for deep learning.
  - `Matplotlib` & `Seaborn` for data visualization.

---

## 🚀 How to Run
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Tawhidrahman292/codealpha_tasks.git
   ```
2. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
   *(Note: Ensure you have Python installed on your system.)*
3. **Execute Scripts**:
   ```bash
   python diabetes_prediction.py
   python mnist_cnn.py
   ```
